ESP32 FM Radio with RDA5807M, LCD, Rotary Encoder and Station Memory

 # ESP32 FM Radio

A complete FM radio built with ESP32 and RDA5807M.

## Features

- FM Radio
- Auto Scan
- Save Station
- Load Station
- Replace Station
- LCD 16x2
- Rotary Encoder
- Boot Screen

## Hardware

- ESP32
- RDA5807M
- PAM8403
- LCD1602 I2C
- KY-040 Encoder
